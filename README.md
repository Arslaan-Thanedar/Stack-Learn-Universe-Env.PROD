## Setup and Run

To run tests, run the following command

```bash
    cd Stack Learn Universe Env.PROD
```
Setup dotnet application : 
```bash
    dotnet run --urls=https://localhost:5001/
```
Once dependencies are build close application and run :

```bash
    dotnet watch
```
